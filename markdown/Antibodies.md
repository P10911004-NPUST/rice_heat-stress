# Antibodies
- Hygromycin B
- G418

## Hygromycin
### 原液
> 棕色原液 (10 mL / 支)  
廠商 : 賽魯士

### Stock solution (50 mg / mL)
以滅菌 ddH<sub>2</sub>O 定量至 20 mL 稀釋成母液 (50 mg / mL) 備用。  
可存放在第四個 -20&deg;C 冰櫃的上 2 左 3 第 4 格 Jhenny 的儲存格

### Solution (50 &micro;g / mL ; 50 ppm)
將母液 (50 mg / mL) 稀釋 1000 倍，即可使用。
