# **Polymerase Chain Reaction ( PCR )**

# Preparation  

| Material <sup>&#8251;</sup> | Concentration  | Addition per sample  | Location                                 |
| :----------:                | :---:          | :---:                | :---:                                    |
| gDNA <sup>&dagger;</sup>    | 10 ng / uL     | 4 &micro;L           | Usually -20&deg;C                        |
| F-primer                    |                | 0.4 &micro;L         | -20&deg;C (2row-3col-4layer)             |
| R-primer                    |                | 0.4 &micro;L         | -20&deg;C (2row-3col-4layer)             |
| 2X-Taq                      |                | 5 &micro;L           | -20&deg;C (4row-3col-4layer; orange lid) |
| Ambion water                |                | 0.2 &micro;L         | RNA bench (right-hand side, top drawer)  |
| Total :                     |                | 10 &micro;L          |                                          |

<sup>&#8251;</sup> All materials should be placed on ice.

<sup>&dagger;</sup> gDNA : The gDNA stock solution of all samples should be standardized to 
the same concentration, for example 10 ng/<i>&micro;</i>L.

<br>
<div style='page-break-after: always'></div>

# Procedure

1. Prepare the total amount of the required working solution ( F-primer, R-primer, 2X-Taq );

> - For example, if we had 10 samples, then me make the working solution by adding 
> 4 &micro;L <b style="color:orange">F-primer</b>, 
> 4 &micro;L <b style="color:orange">R-primer</b>, 
> and 50 &micro;L <b style="color:orange">2X-Taq</b> into a new 200 &micro;L PCR tube.

2. Dispense the working solution into a new 200 <i>&micro;</i>L PCR tubes ( 5.8 &micro;L for each sample );

3. Lastly, add the sample gDNA stock solution;

4. Close the lid tightly, and swipe the tube gently with finger several time to mix the solution well;

5. Slightly centrifuge to bring down the solution;

6. Put into the PCR machine ( the machine should be warmed-up around 2 mins before use ).

## Reaction curve:

| Process                        | Temperature (&deg;C)              | Time        | Cycle  |
| :----------:                   | :---:                             | :---:       | :---:  |
| Initial denaturation           | 95                                | 1 min       | --     |
| Denaturation                   | 95                                | 15 secs     | 30     |
| Annealing<sup>&dagger;</sup>   | <b style='color: red;'>56<b>      | 15 secs     | 30     |
| Extension                      | 72                                | 1 min       | 30     |
| Complete extension             | 72                                | 5 min       | --     |
| PCR product storage            | 4                                 | &#8734;     | --     |
|||||

<sup>&dagger;</sup> Annealing temperature depends on primers.
