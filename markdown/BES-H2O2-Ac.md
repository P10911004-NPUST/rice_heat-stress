# <b>BES &ndash; H<sub>2</sub>O<sub>2</sub> &ndash; Ac</b>

Hydrogen peroxide ( H<sub>2</sub>O<sub>2</sub> ) detection probes.

> Manufacturer : <a href='https://labchem-wako.fujifilm.com/us/product/detail/W01W0102-1781.html'>Wako</a>  
> Molecular Formula : C<sub>28</sub>H<sub>11</sub>F<sub>7</sub>O<sub>8</sub>S  
> M.W. = 640.44 g / mol

Stock solution ( 10 mM ) = 1 mg powder dissolved in 156.2 <i>&micro;</i>L DMSO  
- storage temperature: 25&deg;C

<br>

Working solution: 10 - 50 <i>&micro;</i>M in medium.

For Arabidopsis root: Incubate in room temperature for 10 mins, protect from light.

<br>

Excitation: 485 nm ( 488 nm )  
Emission: 515 nm ( 495 - 535 nm )
